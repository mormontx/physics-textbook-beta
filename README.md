# Digital Physics Textbook

A modern, interactive digital textbook for learning physics.

## Features

- **Interactive Navigation**: Easy-to-use sidebar navigation to explore different physics topics.
- **Rich Visuals**: Engaging design with a galaxy-themed header and toxic-green/solar-punk aesthetic accents.
- **Conceptual Questions**: Dedicated "Think About It" section for conceptual understanding.
- **Equation Derivations**: Step-by-step derivations for key physics equations (coming soon).
- **Responsive Design**: optimized for various screen sizes.

## Technologies Used

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- JavaScript
- MathJax (for equation rendering)
- Google Fonts (Sora)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/mormontx/digital-textbook-project.git
   ```
2. Open `index.html` in your web browser.

## Project Structure

- `index.html`: Main entry point.
- `styles.css`: Custom styling and themes.
- `app.js`: Application logic.
- `solution_manual/`: Resources and additional materials.
